## Quick Start

### Prerequisites
- Python 3.9+
- Google Docs API credentials

### Installation
```bash
git clone https://github.com/badkirked/david-hill.git
cd david-hill
pip install -r requirements.txt
```

### Usage
```bash
python3 create_google_doc.py
```
